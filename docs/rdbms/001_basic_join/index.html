    조인(Join) - 조인 기반 메커니즘 #

    조인 #

    • 2개 이상의 테이블을 서로 연결하여 데이터 추출
    • 관계형 DB에서는 조인을 통해 서로 다른 테이블간의 정보를 원하는 대로 가져올 수 있음

    중요한 부분 #

    • 1:M 조인시, 결과 집합은 M 집합의 레벨을 그대로 유지한다.
    • 조인 결과 : M 집합

    Left Outer Join #

    • Right Outer Join은 사용하지 말자. -> 테이블 위치를 바꾸면 Left Outer join처럼 사용 가능

    Left Outer Join 또다른 표기 (+) # 

    SELECT *
FROM TEST1 A, TEST2 B
WHERE A.COL1 = B.COL1(+)

--

SELECT *
FROM TEST1 A LEFT OUTER JOIN TEST2 B
ON A.COL1 = B.COL1
